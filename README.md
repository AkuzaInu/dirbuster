# Python Dirbuster
`v1.2`

## Description
A python directory buster using `requests`

## Requirements
1. python `3.9.10`
2. requests `2.27.1`
3. colorama `0.4.4`

```
pip install -r requirements.txt
```

## Usage
```
python main.py
```
